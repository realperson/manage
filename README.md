angular 4 版管理系统前端

## 安装

### 安装@angular/cli:
```bash
npm install -g @angular/cli@latest
npm config set sass_binary_site=https://npm.taobao.org/mirrors/node-sass/
```

### 安装依赖包:
```bash
npm install
```

### 在浏览器中运行( `http://localhost:4200/`):
```bash
ng serve
```
